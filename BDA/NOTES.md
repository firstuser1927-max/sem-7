## Practicals (Key ones)
1. MapReduce: Matrix multiply, Sorting, Indexing
2. Distributed Cache, Map-Side Join, Word Count
3. Matrix algorithms in Spark SQL
4. K-Means via MapReduce
5. Frequent Item Set via MapReduce
6. PySpark + Hive: COVID-19 Pipeline

## Exam Strategy
- Units 2, 3, 5 = 60% of exam — these are the core
- Unit 1 is easy marks — know the 5 V's and definitions cold
- Spark (Unit 6) is only 10% but very practical-heavy
