# Social and Information Network Analysis (3174209)

Subject Code : 3174209

Subject Name : Social and Information Network Analysis

| WEF Academic Year : | 2021 - 22 |
| --- | --- |
| Semester : | 7 |
| Category of the Course : | Open Elective |

Prerequisite : Programming language (python etc.), Basic knowledge of algorithm and programming along with AI and graph theory.

Rationale : This course will cover data analysis on social networks, with a particular emphasis on effective methods for managing large-scale networks. It presents the most important theoretical results in social network mining as well as practical experience on the most important challenges pertaining to this field.
