| Unit | Topic | Hours | Weightage |
|------|-------|-------|-----------|

## Practicals (Key ones)
1. Matplotlib — Line Plots
2. Waffle Charts, Word Clouds, Regression Plots
3. Prepare various charts using datasets
4. Prepare data model using PowerBI

## Exam Strategy
- Unit 2 (Matplotlib) = 25% — know all chart types and when to use each
- Units 1, 3, 4 = 20% each — Power BI is increasingly exam-relevant
- Unit 5 (Data Modeling) = 15% — understand ER diagrams, data model types
