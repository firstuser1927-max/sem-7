1. NN model with one hidden layer for classification
2. MLP for prediction (e.g., house prices)
3. CNN for Image Classification
4. RNN/LSTM/GRU implementation
5. ANN on GPUs

## Exam Strategy
- Units 2, 3, 5 carry 75% of exam marks — prioritize these
- Unit 3 (Fuzzy Neural Systems) is unique/tricky — needs dedicated lessons
- Unit 5 has the most algorithms — make a cheat sheet reference
