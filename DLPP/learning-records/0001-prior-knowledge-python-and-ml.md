# Prior Knowledge Established — Python & Basic ML

The learner has working knowledge of Python programming and foundational machine learning concepts (supervised learning, basic model training). This means:

- Deep Learning lessons can skip Python syntax basics and go straight to frameworks (TensorFlow/Keras)
- ML concepts like gradient descent, loss functions, and train/test splits can be assumed without explanation
- NLP lessons can build directly on vectorization concepts (e.g., bags of words) without defining what a feature vector is
- Practicals can assume `import numpy as np` / `import pandas as pd` are familiar patterns

## Implications

- Start all subjects from their **application layer**, not from first principles of programming
- Use analogies to known ML concepts when introducing new deep learning or NLP ideas
- First lesson priority should be **DLPP (Unit 1)** — the 5-credit core subject that bridges ML → Deep Learning
